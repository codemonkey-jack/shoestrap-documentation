The `shoestrap_sanitize_hex` function has been deprecated since Shoestrap 3.2

It has now moved inside the [`Shoestrap_Color`](/classes/Shoestrap_Color) class.

```php
<?php return Shoestrap_Color::sanitize_hex( $color ); ?>
```

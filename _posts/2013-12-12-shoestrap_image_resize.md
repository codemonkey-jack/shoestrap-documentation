The `shoestrap_image_resize` function has been deprecated since Shoestrap 3.2

It has now moved inside the [`Shoestrap_Image`](/classes/Shoestrap_Image) class.

```php
<?php return Shoestrap_Image::image_resize( $data ); ?>
```

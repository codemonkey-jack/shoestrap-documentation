Get ready for a better theme!

You can now submit your own documentations and help us complete this task by forking our documentation repository on github.

We use Jekyll to create this documentation site, so all you have to do in order to contribute to these docs is edit the file that interests you (or add a new file) on the _posts folder and submit a pull request to our github repository. Please note the the files should be formatted using github markdown syntax for better consistency.

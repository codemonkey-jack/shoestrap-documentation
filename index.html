The most versatile Bootstrap theme for WordPress just got better!

We 've been rewriting the Shoestrap theme for more than 2 months, making it better and even more versatile.

As a result of the theme rewrite, a lot of things have changed and the previous docs we had available for developers are obsolete and no longer available.

Please be patient as we rewrite them and bring them up to date.

You can now submit your own documentations and help us complete this task by forking our documentation repository on github.

We use Jekyll to create this documentation site, so all you have to do in order to contribute to these docs is add a new file to the _posts folder and submit a pull request to our github repository. Please note the the files should be formatted using github markdown syntax for better consistency.
